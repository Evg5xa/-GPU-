GPU-ускоренное моделирование акустики помещений


## Описание
Моделирование распространения звука в помещении с учётом частотных характеристик материалов. Реализованы методы трассировки лучей (Ray Tracing) и зеркальных источников (Image Source) с использованием GPU (CUDA) через PyTorch.

## Возможности
- 10 частотных полос (31.5 Гц – 16 кГц)
- Выбор материалов стен с частотно-зависимыми коэффициентами поглощения
- Визуализация энергетических карт для каждой частоты
- Частотные характеристики в точках микрофонов
- Ускорение вычислений в десятки/сотни раз по сравнению с CPU
